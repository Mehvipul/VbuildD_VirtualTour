
VbuildD | Virtual Tour for Pre-Construction Homes

Developed VbuildD, an innovative virtual tour platform tailored for showcasing homes before construction 
commences.
The project aimed to revolutionize the real estate market by providing prospective buyers and investors with immersive 
3D experiences of properties, enabling them to explore and visualize architectural designs and layouts in detail.
Led the creating interactive 3D environments and user interfaces using Unity and creating 3d model in SketchUp.
Significantly boosted user engagement and offering compelling virtual tours, resulting in a 90% increase in lead 
generation.
Optimized performance to ensure smooth navigation and high-quality visuals across various devices and VR headsets.
